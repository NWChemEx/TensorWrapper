Detailed Description

template<typename TileType>
struct tensorwrapper::tensor::Conversion< TA::DistArray< TileType, TA::SparsePolicy > >

Specialization for DistArray conversion cases.

This specialization is intended to return the tensor wrapped by a Buffer back as a TA DistArray. Since we only have the TA backend at the moment, this just grabs a reference to the tensor that the buffer is holding onto.

Template Parameters
TileTypeThe type of the tiles in the DistArray.
