Detailed Description

template<typename FieldType>
class tensorwrapper::tensor::expression::detail_::Subtract< FieldType >

Implements subtraction of two expressions.

This class holds two expressions, referred to as a and b, and computes a - b when evaluated.

Template Parameters
FieldTypeA strong type representing the mathematical field from which the tensors' elements are drawn.
