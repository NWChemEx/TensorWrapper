Detailed Description

template<typename FloatType>
class tensorwrapper::buffer::Eigen< FloatType >

A buffer which wraps an Eigen object.

Template Parameters
FloatTypeThe type used to store the elements of the object.

Right now the backend is always an Eigen Tensor, but concievably it could be generalized to be matrices or Eigen's map class.
