Detailed Description

Common base class for all buffer objects.

All classes which wrap existing tensor libraries derive from this class.

buffer_base_reference tensorwrapper::buffer::BufferBase::permute_assignment ( label_type  this_labels,
const_labeled_buffer_reference  rhs 
)
inline

Sets *this to a permutation of rhs.

rhs.rhs() are the dummy indices associated with the modes of the buffer in rhs and this_labels are the dummy indices associated with the buffer in *this. This method will permute rhs so that the resulting buffer's modes are ordered consistently with this_labels, i.e. the permutation is FROM the rhs.rhs() order TO the this_labels order. This is seemingly backwards when described out, but consistent with the intent of a DSL expression like t("i,j") = x("j,i"); where the intent is to set t equal to the transpose of x.

Parameters
[in]this_labelsthe dummy indices for the modes of *this.
[in]rhsThe tensor to permute.
Returns
*this after setting it equal to a permutation of rhs.
Exceptions
???If the derived class's implementation of permute_assignment_ throws. Same throw guarantee.
