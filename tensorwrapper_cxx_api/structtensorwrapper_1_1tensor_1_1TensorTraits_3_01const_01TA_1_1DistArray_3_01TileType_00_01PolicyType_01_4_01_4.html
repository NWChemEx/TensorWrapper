Detailed Description

template<typename TileType, typename PolicyType>
struct tensorwrapper::tensor::TensorTraits< const TA::DistArray< TileType, PolicyType > >

Specializes TensorTraits for a TiledArray tensor.

This partial specialization of TensorTraits is instantiated when the template type parameter is a const TiledArray tensor.

Template Parameters
TileTypeThe type of the tiles in the TiledArray tensor.
PolicyTypethe type of the policy used by the TiledArray tensor.
