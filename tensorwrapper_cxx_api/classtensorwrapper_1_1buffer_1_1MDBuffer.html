Detailed Description

A multidimensional (MD) buffer.

This class is a dense multidimensional buffer of floating-point values.

◆ MDBuffer() [1/5]

tensorwrapper::buffer::MDBuffer::MDBuffer ( )
defaultnoexcept

Creates an empty multi-dimensional buffer.

The resulting buffer will have a shape of rank 0, but a size of 0. Thus the buffer can NOT be used to store any elements (including treating *this as a scalar). The resulting buffer can be assigned to or moved to to populate it.

auto tensorwrapper::buffer::MDBuffer::get_elem ( index_vector  index) const

Returns the element with the offsets specified by index.

This method will retrieve a const reference to the element at the offsets specified by index. The length of index must be equal to the rank of *this and each entry in index must be less than the extent of the corresponding mode of *this.

This method can only be used to retrieve elements from *this. To modify elements use set_elem().

◆ operator==()

bool tensorwrapper::buffer::MDBuffer::operator== ( const my_type &  rhs) const
noexcept

Compares two MDBuffer objects for exact equality.

Two MDBuffer objects are exactly equal if they have the same shape and if all of their corresponding elements are bitwise identical. In practice, the implementation stores a hash of the elements in the tensor and compares the hashes for equality rather than checking each element individually.

Parameters

Returns
