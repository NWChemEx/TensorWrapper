Detailed Description

template<typename FloatType>
class tensorwrapper::buffer::Contiguous< FloatType >

Denotes that a buffer is held contiguously.

Contiguous buffers are such that given a pointer to the first element p, the i-th element (i is zero based) is given by dereferencing the pointer p + i. Note that contiguous buffers are always vectors and storing higher rank tensors in a contiguous buffer requires "vectorization" of the tensor. In C++ vectorization is usually done in row-major format.
