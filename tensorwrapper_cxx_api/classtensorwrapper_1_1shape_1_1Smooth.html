Detailed Description

Describes the shape of a "traditional" tensor.

Tensors are traditionally thought of as being (hyper-)rectangular arrays of scalars. The geometry of such a shape is described by stating the geometric dimension of the (hyper-)rectangle and the number of elements in the array.

inlinenoexcept

Is *this the same shape as rhs?

Note
This is a non-polymorphic value comparison, i.e., any state in *this or rhs that resides in derived classes is NOT considered in this comparison.

Two Smooth objects are value equal if they contain the same number of modes and if their -th modes have the same extent for all in the range [0, rank()).
