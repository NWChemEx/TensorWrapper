Detailed Description

Dispatches to the appropriate backend based on the FP type.

This visitor is intended to be used with WTF's buffer visitation mechanism. This base class implements the logic common to all binary operations and lets the derived classes implement the operation-specific logic.

Note
This class derives from UnaryOperationVisitor to reuse some of its functionality. This inheritance is private because it does not make sense to use a BinaryOperationVisitor as a UnaryOperationVisitor.
