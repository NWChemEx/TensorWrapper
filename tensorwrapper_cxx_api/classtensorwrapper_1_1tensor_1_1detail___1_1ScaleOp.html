Detailed Description

template<typename FloatType, typename TensorType>
class tensorwrapper::tensor::detail_::ScaleOp< FloatType, TensorType >

Implements scaling of a tensor in the operation layer.

This class is charged with determining what:

<expr> * scalar

or

<scalar> * <expr>

works out to. More specifically given am expression and a scalar this class is charged with scaling the tensor.

Template Parameters
FloatTypeThe type of the scalar;
TensorTypeThe type of the expression.
