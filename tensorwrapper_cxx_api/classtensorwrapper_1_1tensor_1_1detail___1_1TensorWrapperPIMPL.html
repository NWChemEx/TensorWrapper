Detailed Description

template<typename FieldType>
class tensorwrapper::tensor::detail_::TensorWrapperPIMPL< FieldType >

Holds the state of a TensorWrapper.

The TensorWrapperPIMPL class is designed to encapsulate as much of the TensorWrapper state as possible. Full encapsulation is not possible unless we want to supress the expression templating of the wrapped library. For TiledArray this "leakage" occurs through the annotate member (and also somewhat through the ctor).

Template Parameters
FieldTypeWhat sort of elements are in the tensor? Assumed to be either field::Scalar or field::Tensor.
