template<typename LHSType, typename RHSType>
class tensorwrapper::tensor::detail_::SubtOp< LHSType, RHSType >

Implements the operator- of the expression layer.

Template Parameters

RHSTypeThe type on the right side of the subtraction operator. Assumed to be a class derived from OpLayer.

Syntactic sugar for generating a SubtOp instance.

These four overloads of operator- allow you to subtract tensor wrappers of different const-ness via the expression layer.
