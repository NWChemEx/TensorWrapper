Detailed Description

Namespace for utilities that generate synthetic tensors and matrices.

The functions in this namespace create reproducible test matrices, eigenvalue spectra, and related quantities for unit testing and benchmarking.

Adds clamped normal noise to each element of matrix.

Draws delta ~ Normal(0, t) and clamps to [-t, t] before adding to each element. When t is zero the input is copied unchanged.

For sigma uncertain types the result has standard deviation t. For sigma interval types the result has radius t about the perturbed center.

Explicit instantiations are provided only for types in types::floating_point_types.

std::invalid_argumentif t is negative.
