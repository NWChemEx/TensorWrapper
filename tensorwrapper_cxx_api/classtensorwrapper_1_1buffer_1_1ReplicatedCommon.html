Detailed Description

template<typename Derived>
class tensorwrapper::buffer::ReplicatedCommon< Derived >

Implements and defines common functionality for replicated buffers.

Template Parameters
DerivedThe class *this is implementing. Expected to be unqualified Replicated or ReplicatedView.

To use this class the derived class must define:

  • const_element_reference get_elem_(index_vector index) const so that it returns the element at the given index.
  • void set_elem_(index_vector index, element_type value) so that it sets the element at the given index to the given value.

This class is used to implement the common functionality for replicated buffers.
