Detailed Description

template<typename... Args>
struct tensorwrapper::tensor::detail_::LabeledVariant< std::variant< Args... > >

Specializes LabeledVariant for std::variant.

This specialization determines the type that results from labeling/annotating every tensor in the input variant. The resulting variant type is stored as the member type type.

Template Parameters
ArgsThe types in the std::variant. The types are assumed to be TA::DistArray instantiations.
