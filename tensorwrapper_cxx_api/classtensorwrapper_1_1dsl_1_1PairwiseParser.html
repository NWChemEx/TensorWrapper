Detailed Description

template<typename ObjectType, typename LabelType>
class tensorwrapper::dsl::PairwiseParser< ObjectType, LabelType >

Object which evaluates the AST of an expression pairwise.

Template Parameters
ObjectTypeThe type of the objects associated with the dummy indices. Expected to be possibly cv-qualified versions of Tensor, buffers, shapes, etc.
LabelTypeThe type of object used for the dummy indices.

The easiest way to evaluate an abstract syntax tree which contains operations involving at most two objects is by splitting it into subtrees which contain at most two connected nodes, i.e., considering each operation pairwise. That's what this parser does.

Recursion end-point.

Evaluates rhs given that it will be evaluated into lhs. This is the natural end-point for recursion down a branch of the AST.

N.b., this overload is only responsible for evaluating rhs NOT for assigning it to lhs.
