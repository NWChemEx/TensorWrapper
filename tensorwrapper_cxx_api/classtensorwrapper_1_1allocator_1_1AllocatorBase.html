Detailed Description

Common base class for all allocators.

The AllocatorBase class serves as type-erasure and a unified API for all allocators.

◆ allocate()

buffer_base_pointer tensorwrapper::allocator::AllocatorBase::allocate ( layout_pointer  playout)
inline

Polymorphically allocates a new buffer.

This method type-erases the process of creating a buffer by dispatching to the derived class. In general the buffer created by this method will NOT be initialized, though this will depend on the default behavior of the backend. Use construct instead of allocate if you additionally want to guarantee initialization.

Derived classes implement this method by overriding allocate_.

Parameters
[in]playoutA pointer to the layout for the new buffer.
Returns
The newly allocated, but not necessarily initialized buffer.

bool tensorwrapper::allocator::AllocatorBase::operator== ( const AllocatorBase &  rhs) const

Is *this value equal to rhs?

This method is non-polymorphic and only compares the AllocatorBase part of *this to the AllocatorBase part of rhs. Two AllocatorBase objects are value equal if they contain views of the same runtime.
