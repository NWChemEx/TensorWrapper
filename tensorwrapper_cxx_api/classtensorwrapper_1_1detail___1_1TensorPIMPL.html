Detailed Description

Actually implements a Tensor object.

At a high-level Tensors have two things: a logical layout and a buffer. This class wraps that state together.

Is *this value equal to rhs?

Two TensorPIMPL objects are value equal if their respective logical layouts and buffers compare value equal. Value equality of the layout and the buffer are done polymorphically.
