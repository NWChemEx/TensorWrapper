template<typename ReplicatedType>
class tensorwrapper::buffer::ReplicatedView< ReplicatedType >

A view of a Replicated buffer.

Template Parameters
ReplicatedTypeThe type of the Replicated buffer to view. Expected to be unqualified Replicated or const Replicated.

This class is a view of a Replicated buffer. It is used to create a view of a Replicated buffer.
