Detailed Description

template<typename Derived>
class tensorwrapper::buffer::BufferBaseCommon< Derived >

CRTP base factoring the common layout/equality API of BufferBase and BufferViewBase.

Derived must implement the protected hooks: has_layout_(), layout_(), and approximately_equal_(const BufferBase&, double).

Template Parameters
DerivedThe CRTP derived type (BufferBase or BufferViewBase).
