Detailed Description

Works out the details pertaining to an arbitrary binary einsum op.

For a general einsum operation the indices in a label fall into one of four categories:

  • trace indices: appear in only one of the input tensors, but not the output
  • dummy indices: those that appear in both input tensors, but not the output
  • free indices: appear in result and ONE of the input tensors
  • batch indices: appear in all three tensors

N.b., though the set of indices in say lhs_batch and rhs_batch must be the same, the order can be different. This applies to dummy indices too.
