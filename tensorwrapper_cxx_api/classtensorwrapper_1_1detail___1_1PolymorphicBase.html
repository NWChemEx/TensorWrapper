Detailed Description

template<typename BaseType>
class tensorwrapper::detail_::PolymorphicBase< BaseType >

Defines the API polymorphic utility methods should use.

Template Parameters
BaseTypethe base class of the hierarchy being implemented.

The PolymorphicBase class is designed to make it easier to write a polymorphic object hierarchy by providing standardized APIs and default implementations for those functions.

Determines if *this and rhs are polymorphically equal.

Calling operator== on an object of type T is supposed to compare the state defined in class T as well as all state defined in parent classes. If there is other classes which derived from T that possess state, use of T::operator== will not consider such state in the comparison. This method casts both *this and rhs to their most derived class and then performs the value comparison to ensure that all state is considered. If *this and rhs have different most derived classes this comparison returns false.

Derived classes should override are_equal_ to implement this method.

Parameters
[in]rhsThe object to compare to.
Returns
True if *this and rhs are polymorphically value equal and false otherwise.
Exceptions
NoneNo throw guarantee.

Creates a deep polymorphic copy of *this via the base class.

Calling the copy constructor of an object of type T is supposed to return a deep copy of the a T object. When T is polymorphic such a copy slices off the pieces of the object defined in classes which derive from T. Calling clone will ensure that the entire object is copied, including the pieces in derived classes.

Derived classes should override clone_ to implement this method.

Returns
A deep polymorphic copy of *this.
Exceptions
std::bad_allocif there is a problem allocating the copy. Strong throw guarantee.
