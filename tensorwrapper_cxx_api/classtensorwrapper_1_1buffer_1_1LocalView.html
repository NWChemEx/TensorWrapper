Detailed Description

template<typename LocalType>
class tensorwrapper::buffer::LocalView< LocalType >

A view of a Local buffer.

This class is a view of a Local buffer. It is used to create a view of a Local buffer. It is not a strong type and does not impart any additional state to the BufferViewBase class.
