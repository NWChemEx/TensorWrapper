Detailed Description

Class for working out details pertaining to a tensor contraction.

N.B. Contraction covers direct product (which is a special case of contraction with 0 dummy indices).

label_type tensorwrapper::buffer::ContractionPlanner::result_matrix_labels ( ) const
inline

Flattened result labels.

After applying lhs_permutation to LHS to get A, and rhs_permutation to RHS to get B, A and B can be multiplied together with a gemm. The resulting matrix has indices given by concatenating the free indices of A with the free indices of B. This method returns those indices.
