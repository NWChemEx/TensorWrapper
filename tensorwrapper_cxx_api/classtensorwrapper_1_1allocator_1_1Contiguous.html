contiguous_pointer tensorwrapper::allocator::Contiguous< FloatType >::allocate ( const_layout_reference  layout)
inline

Allocates a contiguous pointer given layout.

Note
These methods shadow the function of the same name in the base class. The intent is to avoid needing to rebind a freshly allocated buffer when the user already knows it is a Contiguous buffer.
Parameters
[in]layoutThe layout of the tensor to allocate. May be passed as a unique_ptr or by reference. If passed by reference will be copied.
Returns
A pointer to the newly allocated buffer::Contiguous object.
