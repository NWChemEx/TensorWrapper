Detailed Description

template<typename LHSType, typename RHSType>
class tensorwrapper::tensor::detail_::AddOp< LHSType, RHSType >

Implements operator+ for the operation layer.

This class is charged with determining what:

<expr> + <expr>

works out to. More specifically given two pieces of an expression this class is charged with adding the variants contained in those pieces.

Template Parameters
LHSTypeThe type of the object on the left side of operator+
RHSTypeThe type of the object on the right side of operator+
