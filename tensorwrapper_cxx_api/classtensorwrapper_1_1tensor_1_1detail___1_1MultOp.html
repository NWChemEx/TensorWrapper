template<typename LHSType, typename RHSType>
class tensorwrapper::tensor::detail_::MultOp< LHSType, RHSType >

Implements the operator* of the expression layer.

Template Parameters

RHSTypeThe type on the right side of the multiplication operator. Assumed to be a class derived from OpLayer.
