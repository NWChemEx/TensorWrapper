Detailed Description

template<typename T, std::size_t N>
struct tensorwrapper::tensor::NDInitializerListTraits< T, N >

Traits class for an N -dimensional initializer list of type T.

This traits class is primarily envisioned as being used with the TensorWrapper class in order to deduce the type of an initializer list for a rank N tensor. Unrolling the first few results:

using rank0_il = T;
using rank1_il = std::initializer_list<T>;
using rank2_il = std::initializer_list<std::initializer_list<T>>;

Bascially a rank N tensor requires N nested initializer lists.

This class is the primary template and works out the initializer list when N is not 0. For N not equal to 0, the type of an initializer list for a rank N tensor is an initializer_list of rank N - 1 initializer_lists. The type of the initializer list for a rank N tensor is accessible via the member type type.

Template Parameters
TThe scalar type in the tensor. This will be the type the most nested initializer_list is filled with.
NAn unsigned integer specifying the rank of the tensor the initializer_list is for.
