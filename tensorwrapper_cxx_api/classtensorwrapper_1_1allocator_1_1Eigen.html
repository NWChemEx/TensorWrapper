Detailed Description

template<typename FloatType>
class tensorwrapper::allocator::Eigen< FloatType >

Used to allocate buffers which rely on Eigen tensors.

Template Parameters
FloatTypeThe numerical type the buffer will use to store the elements.
RankThe rank of the tensor stored in the buffer.

This allocator is capable of creating buffers with Eigen tensors in them.

Rebinds a buffer to the same type as *this.

This method will convert buffer into a buffer which could have been allocated by *this. If buffer was allocated as such a buffer already, then this method is simply a downcast.
