Detailed Description

template<typename FloatType, unsigned short Rank>
class tensorwrapper::allocator::Eigen< FloatType, Rank >

Used to allocate buffers which rely on Eigen tensors.

Template Parameters
FloatTypeThe numerical type the buffer will use to store the elements.
RankThe rank of the tensor stored in the buffer.

This allocator is capable of creating buffers with Eigen tensors in them.

◆ allocate() [2/2]

template<typename FloatType , unsigned short Rank>
eigen_buffer_pointer tensorwrapper::allocator::Eigen< FloatType, Rank >::allocate ( eigen_layout_pointer  playout)

Primary method for allocating Eigen-based buffers.

This method simply checks that playout points to a layout compatible with the template parameters of *this and then creates a new Eigen tensor object. The elements of the Eigen tensor object are NOT initialized.

Parameters
[in]playoutA pointer to the layout for the new buffer.
Returns
An uninitialized buffer containing playout.
Exceptions
std::runtime_errorif the provided layout does not have the same rank as Rank. Strong throw guarantee.
std::bad_allocif there is a problem allocating the new buffer. Strong throw guarantee.
